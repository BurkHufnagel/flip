Base classes used to integrate Flip to the web.

If you want to process the request attributes and to store them in the session to override the features defines in the standard way add FlipFilter to your web.xml. The code snipped used to add it in the web.xml is: 

<filter>
  <filter-name>flip</filter-name>
  <filter-class>com.tacitknowledge.flip.servlet.FlipFilter</filter-class>
</filter>

<filter-mapping>
  <filter-name>flip</filter-name>
  <url-pattern>*.jsp</url-pattern>
</filter-mapping>
