Tags used in JSP pages to render the content depending on the state of the feature. To be able to use the Flip JSP tags you should add to your web.xml the following snippet: 
<taglib>
  <taglib-uri>http://www.tacitknowledge.com/flip</taglib-uri>
</taglib>
and in your JSP declare it as: 
<%@ taglib prefix="flip" uri="http://www.tacitknowledge.com/flip" %>

Usage

flip:enabled and flip:disabled

This tag is used to render the content of the tag depending of the feature state. These tags has the following attributes:
feature
declares the name of the feature to evaluate. If the tag is flip:enabled then the content will be rendered only if the feature has a state as ENABLED. The same rule applied to the tag flip:disabled, the feature should have the sate as DISABLED.
service
sets the FeatureService used to evaluate the feature. If this attribute is not specified then the tag will try to obtain the instance of the feature service from request attribute featureService and then from FlipWebContext#getFeatureService().

flip:sessionFeatures

This tag is used to display the current feature descriptors found in the session. In the session are stored the feature descriptors which was obtained from request parameters by FlipOverrideFilter. It displays the features in the format: 
<feature-name> = <feature-state><br />
